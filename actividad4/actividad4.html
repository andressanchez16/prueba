Section

Section se usa para estructurar y dividir tu página web. Determina qué contenido corresponde a qué parte de un esquema

Div

Div es la etiqueta que determina las divisiones dentro de section

Main

Abarca el contenido principal de la página web en body (solo se puede poner una vez)

Article

Article sirve para contener información que puede dividirse de las demás etiquetas

Aside

Sección relacionada directamente con el contenido principal (se usa con flexbox para poner diferentes secciones o columnas)
