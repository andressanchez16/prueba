¿Qué es Css?

CSS es cascade style sheets que sirve para poner propiedades decorativas en una página web de html

¿Qué es una referencia por palabra clave?

Es cuando en un documento css pones el nombre de una etiqueta para que se le apliquen propiedades decorativas a dicha etiqueta seleccionada

¿Qué es una referencia por id?

En un documento HTML, los selectores de ID de CSS buscan un elemento basado en el contenido del atributo id. El atributo ID del elemento seleccionado debe coincidir exactamente con el valor dado en el selector. se representa con un #

¿Qué es referencia por clase?

En un documento HTML, los selectores de clase buscan un elemento basado en el contenido de su atributo class . El atributo class está definido como una lista de elementos separados por espacio, y uno de esos elementos debe coincidir exactamente con el nombre de clase dado en el selector. Se representa con un .
